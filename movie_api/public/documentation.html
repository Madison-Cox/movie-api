Movie Finder Documentation

This is a API that will be used on server-side that can be used to retrieve information about many different movies.

Business Logic URL METHOD Request Response
Get genre information /movies/:genre/genreName GET None A JSON object holding data about genre
Read about App / Get None Information about application
Get list of users /users GET None A JSON holding data about all users
Get User by username /users/:Username GET None A JSON object holding data about a single user
Create User /users POST A JSON object holding data about
the user to add
{ "name": "User", "favoriteMovies": [] }		 A JSON object holding data was added
and given an ID
{ "name": "User", "favoriteMovies": [], "id": "3127dcc1-4dbb-410d-b815-e846a0559dfe" }
Update User /users/:Username PUT None Confirmation message verifying
user was added
Delete User /users/:Username/ DELETE None Confirmation message verifying user
was deleted
Get a list of movie /movies GET None A JSON holding data about all movies
Get Director of movie /movies/director/:directorName GET None A JSON object holding data
about a director
Get Genre of movie /movies/Genre/:genreName GET None Returns object holding data about Genre
Get Movie by title /movies/:title GET None Returns object holding data about
specific movie
{ "title": "Top Gun", "Director": { "Name": "Tony Scott" }, "stars": "Tom Cruise, Tim Robbins, Kelly McGillis", "Genre": { "name": "Action" } }
Add Movie to favorites /users/:Username/movies/:MovieID POST None A JSON object holding data was
added and given an ID
Delete Movie /users/:Username/movies/:MovieID DELETE None Confirmation message verifying movie
was removed
